Server Observer

Simple and Reliable Server Monitoring

Server Observer aims to build simple and reliable server monitoring software at an affordable price. Over the next year details will be reveled as the primary development phase takes place.

With Server Observer you will be able to install a small program onto your Linux server which will automaticly reports to our service with vital system statistics. Server Observer will then be able to notify you if you server goes down or if any resources reach a user–defined critical level.
